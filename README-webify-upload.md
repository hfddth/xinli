# Mind Explorer Webify 上传说明

这套项目已经整理成适合腾讯云 Webify 直接上传的静态站点版本。

## 1. 这版适合怎么部署

适合使用腾讯云 Webify 的以下方式：

- 上传本地项目文件夹
- 上传 ZIP 压缩包
- 后续如需要，也可以再接入 Git 仓库持续部署

## 2. 你在 Webify 里只需要做什么

1. 新建一个 Webify 应用
2. 选择上传本地文件或 ZIP 包
3. 上传我整理好的 Webify 部署包
4. 等待平台生成默认域名
5. 打开默认域名测试访问

如果控制台要求填写构建参数，建议使用：

- 安装命令：留空
- 构建命令：留空
- 构建产物目录：`.`
- 部署路径：`/`

## 3. 这版为什么适合 Webify

- 全部是静态文件
- 页面资源使用相对路径
- 不依赖本地开发服务器
- 首页与后台页都是独立 HTML 文件
- 可直接作为静态网站发布

## 4. 当前默认行为

- 如果 `supabase-config.js` 仍为空，网站会以“离线演示模式”运行
- 这时问卷可正常访问、作答、生成报告
- 但不会真正写入数据库

## 5. 如果你只是想先上线给别人访问

直接上传即可。

## 6. 如果你后面还要正式收数

后续再补这三步：

1. 配置数据库
2. 填写 `supabase-config.js`
3. 部署后台导出函数

## 7. Webify 侧建议

- 应用访问根路径保持 `/`
- 首次更新后如果页面没有立即刷新，等待 CDN 缓存更新几分钟再试
- 如果后面要绑定自己的域名，再处理自定义域名和备案
